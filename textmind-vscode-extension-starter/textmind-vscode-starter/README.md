# TEXT MIND VS Code Extension

AI-powered test generation for banking applications directly in VS Code.

## Features

- 🎯 Generate comprehensive tests from JIRA stories
- 🧠 AI-powered scenario generation with RAG
- ✅ Human-in-loop approval workflow
- 📊 Usage tracking and billing dashboard
- 🏦 Banking domain expertise (SOX, PCI-DSS, Reg E)

## Quick Start

1. Install extension from VS Code Marketplace
2. Get API key from https://app.textmind.ai
3. Configure: `Ctrl+Shift+P` → "TEXT MIND: Configure API Key"
4. Generate tests: `Ctrl+Shift+P` → "TEXT MIND: Generate Tests from JIRA"

## Requirements

- VS Code 1.80.0 or higher
- TEXT MIND account (https://app.textmind.ai)
- JIRA instance

## Extension Settings

- `textmind.apiKey`: Your TEXT MIND API key
- `textmind.jiraUrl`: Your JIRA instance URL
- `textmind.apiEndpoint`: TEXT MIND API endpoint (default: https://api.textmind.ai/v1)
